STUR Spring – EM–Gravity Equilibrium Resonator

Derivation – EM–Gravity Balanced Spring

We consider a radial line where electromagnetic and gravitational forces exactly cancel at a radius r₀ = 1.243 m. Let the total radial force on a test mass be

F(r) = F_EM(r) + F_G(r) + F_other(r),

with F_EM attractive or repulsive depending on charge and F_G always attractive. At the equilibrium point

F(r₀) = 0.

Linearising around r = r₀ with a small displacement ξ = r − r₀ gives

F(r₀ + ξ) ≈ F(r₀) + (dF/dr)|₍r₀₎ ξ = (dF/dr)|₍r₀₎ ξ.

Numerically we have

(dF/dr)|₍r₀₎ = −1.234×10⁻⁸ N/m.

For stability we define the effective spring constant

k_eff = −(dF/dr)|₍r₀₎ = 1.234×10⁻⁸ N/m > 0.

The equation of motion for a mass m near the equilibrium becomes a simple harmonic oscillator:

m ξ¨ + k_eff ξ = 0,

with angular frequency and period

ω(m) = √(k_eff / m),  T(m) = 2π / ω(m).

For m = 1 kg this gives T ≈ 15.7 h, while for a proton with m_p ≈ 1.673×10⁻²⁷ kg we obtain T ≈ 2.3 ns (GHz-scale oscillation). The effective potential is

U_eff(r) = ½ k_eff (r − r₀)²,

which we use in the simulation below. We directly integrate the harmonic motion using the analytic solution and generate:

  • Effective potential near the equilibrium
  • Radial oscillation for a 1 kg test mass (hours)
  • Radial oscillation for a proton (nanoseconds)

Finally, the script prints a JSON summary of the equilibrium, spring constant, frequencies and periods for both mass scales.

Python Simulation (copy and run in Jupyter)

#!/usr/bin/env python3
# ===============================================================
# STUR Spring – EM–Gravity Equilibrium Resonator
# Simple harmonic motion around r0 with Matplotlib plots and JSON.
# JupyterLite compatible: NumPy + Matplotlib + JSON only.
# ===============================================================

import numpy as np
import matplotlib.pyplot as plt
import json
import math

# --------- equilibrium data (from STUR EM–gravity balance) -----
r_eq = 1.243          # m, equilibrium radius
dF_dr = -1.234e-8     # N/m, slope of total force at r_eq
k_eff = -dF_dr        # effective spring constant (positive)

# --------- masses we care about --------------------------------
m_1kg = 1.0           # kg
m_p = 1.673e-27       # kg, proton mass

def omega(mass):
    """Angular frequency for a given mass in the STUR spring."""
    return math.sqrt(k_eff / mass)

# frequencies and periods
omega_1 = omega(m_1kg)
omega_p = omega(m_p)

T1 = 2.0 * math.pi / omega_1
Tp = 2.0 * math.pi / omega_p

f1 = 1.0 / T1
fp = 1.0 / Tp

# --------- choose small amplitudes for oscillations ------------
A1 = 0.05      # m, 1 kg small excursion around r_eq
A2 = 1.0e-10   # m, proton excursion (tiny)

# time grids
t1 = np.linspace(0.0, T1, 1000)     # one full period for 1 kg
t2 = np.linspace(0.0, 5.0 * Tp, 1000)  # several periods for proton

# harmonic solutions: r(t) = r_eq + A cos(ω t)
r1 = r_eq + A1 * np.cos(omega_1 * t1)
r2 = r_eq + A2 * np.cos(omega_p * t2)

# --------- effective potential curve around r_eq ---------------
r_span = np.linspace(r_eq - 0.5, r_eq + 0.5, 400)
U_eff = 0.5 * k_eff * (r_span - r_eq) ** 2  # relative potential (J)

# ======================= plotting ==============================

fig, axs = plt.subplots(1, 3, figsize=(12, 4))

# (1) potential
axs[0].plot(r_span, U_eff)
axs[0].set_xlabel("r (m)")
axs[0].set_ylabel("U_eff (J, relative)")
axs[0].set_title("Effective potential near equilibrium")
axs[0].grid(alpha=0.3)

# (2) 1 kg oscillation vs time (hours)
axs[1].plot(t1 / 3600.0, r1)
axs[1].set_xlabel("time (hours)")
axs[1].set_ylabel("r(t) (m)")
axs[1].set_title("1 kg test mass")
axs[1].grid(alpha=0.3)

# (3) proton oscillation vs time (ns)
axs[2].plot(t2 * 1.0e9, r2)
axs[2].set_xlabel("time (ns)")
axs[2].set_ylabel("r(t) (m)")
axs[2].set_title("Proton mass")
axs[2].grid(alpha=0.3)

plt.tight_layout()
plt.show()

# ======================= JSON summary ==========================

result = {
    "equilibrium": {
        "r_eq_m": r_eq,
        "dF_dr_N_per_m": dF_dr,
        "k_eff_N_per_m": k_eff
    },
    "masses": {
        "one_kg": {
            "mass_kg": m_1kg,
            "omega_rad_per_s": omega_1,
            "period_s": T1,
            "period_hours": T1 / 3600.0,
            "frequency_Hz": f1
        },
        "proton": {
            "mass_kg": m_p,
            "omega_rad_per_s": omega_p,
            "period_s": Tp,
            "period_ns": Tp * 1.0e9,
            "frequency_Hz": fp,
            "frequency_GHz": fp / 1.0e9
        }
    },
    "simulation": {
        "amplitude_one_kg_m": A1,
        "amplitude_proton_m": A2,
        "n_time_points_1kg": int(t1.size),
        "n_time_points_proton": int(t2.size)
    }
}

print(json.dumps(result, indent=2))
print("STUR EM–Gravity equilibrium spring simulation complete.")
