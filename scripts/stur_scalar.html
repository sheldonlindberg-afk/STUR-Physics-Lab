STUR Bounce Energy – Scalar Field Cosmological Model

Full Derivation

This simulation models a **cosmological bounce** in the early universe under a scalar field φ(t) coupled to the Unified Resistance framework. We start from the Friedmann equation (Planck units: c=ℏ=8πG=1):

H² = (ρ / 3) (1 - ρ / ρ_c)²,

where ρc is the **critical density** defining the bounce threshold. For a canonical scalar field,

ρ = ½ φ̇² + V(φ), p = ½ φ̇² - V(φ),

and the Klein–Gordon equation governs φ:

φ̈ + 3Hφ̇ + V'(φ) = 0.

Choosing a quadratic potential V(φ) = V₀ φ², the system evolves as:

  • Modified Hubble evolution: dH/dt = −½ φ̇² (1 − 4r + 3r²), where r = ρ/ρ_c.
  • Scale factor acceleration: ä = a (Ḣ + H²).
  • Scalar field motion: φ̈ = −3Hφ̇ − 2V₀φ.

The bounce occurs when ρ = ρ_c → H = 0 → universe reverses from contraction to expansion.

Python Simulation (JupyterLite-Safe)

#!/usr/bin/env python3
# ===============================================================
# STUR Bounce Energy – Scalar Field Cosmological Model
# ===============================================================
import numpy as np, matplotlib.pyplot as plt, json
from scipy.integrate import odeint

# --- Constants (Planck units: c=ℏ=8πG=1) -----------------------
rho_c = 1.0          # Critical density for bounce
V0 = 0.5             # Potential parameter (V = V0 * φ²)
phi0, phidot0 = 0.0, 1.0  # Initial field and velocity

# --- Initial energy and Hubble rate -----------------------------
rho_init = 0.5 * phidot0**2 + V0 * phi0**2
r_init = rho_init / rho_c
rho_eff = rho_init * (1 - r_init)**2
H_init = -np.sqrt(rho_eff / 3)  # negative: contracting phase
a0, adot0 = 1.0, H_init * 1.0

# --- Dynamical equations ---------------------------------------
def equations(state, t):
    a, adot, phi, phidot = state
    if a <= 0: return [0,0,0,0]
    H = adot / a
    rho = 0.5 * phidot**2 + V0 * phi**2
    r = rho / rho_c
    if r >= 1:  # at bounce
        return [0,0,0,0]
    dH_dt = -0.5 * phidot**2 * (1 - 4*r + 3*r**2)
    d_adot_dt = a * (dH_dt + H**2)
    d_phidot_dt = -3*H*phidot - 2*V0*phi
    return [adot, d_adot_dt, phidot, d_phidot_dt]

# --- Time integration ------------------------------------------
t = np.linspace(-10, 10, 1000)
initial_state = [a0, adot0, phi0, phidot0]
solution = odeint(equations, initial_state, t)

# --- Extract solutions -----------------------------------------
a = solution[:,0]
H = solution[:,1] / a
rho = 0.5 * solution[:,3]**2 + V0 * solution[:,2]**2

# --- Plots ------------------------------------------------------
fig, axs = plt.subplots(3,1,figsize=(8,10))
axs[0].plot(t, a, color='cyan')
axs[0].set_ylabel('Scale Factor a(t)')
axs[0].set_title('Cosmological Bounce')

axs[1].plot(t, H, color='magenta')
axs[1].set_ylabel('Hubble H(t)')

axs[2].plot(t, rho, color='orange')
axs[2].set_ylabel('Energy Density ρ(t)')
axs[2].set_xlabel('Time t')

plt.tight_layout()
plt.show()

# --- JSON Summary -----------------------------------------------
summary = {
    "model": "STUR Bounce Energy – Scalar Field Cosmology",
    "rho_c": rho_c,
    "V0": V0,
    "initial_conditions": {"phi0": phi0, "phidot0": phidot0},
    "rho_init": rho_init,
    "H_init": H_init,
    "a_min": float(np.min(a)),
    "a_max": float(np.max(a)),
    "rho_peak": float(np.max(rho)),
    "H_min": float(np.min(H)),
    "H_max": float(np.max(H)),
}
print(json.dumps(summary, indent=2))
print("STUR Bounce Energy simulation complete.")
